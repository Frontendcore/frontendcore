Fixed width

Use _fw to set icons at a fixed width. Great to use when different icon widths throw off alignment. Especially useful in things like nav lists & list groups.

  • Home
  • Book
  • Pencil
  • Cog 
<nav class="navigation _vertical">
    <ul>
        <li>
            <i class="icon-home _fw"></i> Home
        </li>
        <li>
            <i class="icon-book _fw"></i> Book
        </li>
        <li>
            <i class="icon-pencil _fw"></i> Pencil
        </li>
        <li>
            <i class="icon-cog _fw"></i> Cog
        </li>
    </ul>
</nav>

Animated icons

Use the _spin class to get any icon to rotate, and use _pulse to have it rotate with 8 steps. Works well with icon-spinner, icon-refresh, and icon-cog.

<div class="fz-xxl">
    <i class="icon-spinner _spin"></i>
    <i class="icon-circle-o-notch _spin"></i>
    <i class="icon-refresh _spin"></i>
    <i class="icon-cog _spin"></i>
    <i class="icon-spinner _pulse"></i>
</div>
