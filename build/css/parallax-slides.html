Parallax slides

Simple parallax

Parallax component is based on a mixin so you must include it to use it

@include parallax-slides('classname',(height-1,height-2...),('url-1.jpg','url-2.jpg'..));

Settings

classname

First parameter of the mixin. Name of the class.

DEFAULT: slide

(height-1, height-2,...)

List separated by comma of the heights defined in css px for each slide. Example for two slides: (300px,300px)

DEFAULT: empty

('url-1.jpg','url-2.jpg'..)

List separated by comma of the url of the background images for each slide. Example for two slides: ('url-1.jpg','url-2.jpg')

DEFAULT: empty
