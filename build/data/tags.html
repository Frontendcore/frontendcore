Tags input

Input for add tags with autocomplete and great UX experience

	            <input name="tags"  data-tc-modules="tags" value="foo" data-tc-values="foo,baz,bar" />

Settings

value

Value of the input. Use it also to prefill the input with tags.

data-tc-values

List separated by comma of the words and phrases to autocomplete.

data-tc-max

Max number of tags allowed.

DEFAULT: 'false'

placeholder

Text that will be displayed when is empty.

DEFAULT: 'Add a tag...'

data-tc-text-noresult

Text used when there is no results.

DEFAULT: 'No Results Found'

data-tc-text-max

Text used when no more tags are allowed

DEFAULT: 'No More Selections Are Allowed'
