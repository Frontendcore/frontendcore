WYSIWYG

Convert textarea fields or other html elements to editor instances.

	            <textarea name="mytextarea" data-tc-modules="wysiwyg" required></textarea>

Settings

data-tc-plugins

Plugins to load

DEFAULT: ['advlist autolink lists link image charmap print preview anchor pagebreak','searchreplace visualblocks code fullscreen spellchecker searchreplace','insertdatetime media table paste']

data-tc-language

Defines the language of the plugin and the spellchecker.

DEFAULT: 'en'

data-tc-invalid

Invalid html elements.

DEFAULT: 'font,h1,h4,h5,h6,span'

data-tc-spellchecker

Spellchecker ajax url.

DEFAULT: 'http://' + document.domain + '/spellchecker'

data-tc-toolbar

Order of buttons for toolbar 1.

DEFAULT: 'styleselect | bold italic underline | alignleft aligncenter alignright alignjustify | bullist numlist | image media | link | pagebreak | searchreplace | undo redo | spellchecker | code'

data-tc-toolbar-2

Order of buttons for toolbar 2.

DEFAULT: 'false'

data-tc-maxHeight

Max Height(px) for the autoresize plugin

DEFAULT: '600'

data-tc-extended-valid-elements

Extended Valid html elements.

DEFAULT: '-strong,-h2,-h3,-span,-i,-em,-p abbr[title|lang],acronym[title|lang],code[class],object[*],embed[*],param[*]'

data-tc-encoding

controls how entities/characters get processed ('named', 'numeric' or 'raw')

DEFAULT: 'raw'

data-tc-css

Path for the css inside the Wysiwyg iframe content.

DEFAULT: oGlobalSettings.sPathJs + '/libs/tinyMCE/skins/lightgray/content.min.css'
