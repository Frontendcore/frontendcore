Sortable

Drag & Drop sortable lists with ajax callback to save changes.

  • Item 1
  • item 2
  • item 3
<ul class="list" data-tc-modules="sortable" data-tc-defer="distance:50;" data-tc-url="#" data-tc-object="sort"><li id="sortable-1">Item 1</li><li id="sortable-2">item 2</li><li id="sortable-3">item 3</li></ul>

Settings

data-tc-url

Url of the ajax service to save changes.

DEFAULT: 'false'

data-tc-handle

Define it if you want just one element as handle.

DEFAULT: 'none'

data-tc-object

Name of the object array that should be send to the ajax. (Example &object[1]=sortable-1)

DEFAULT: 'sortable'
