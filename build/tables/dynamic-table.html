Attributes

data-tc-pagination

If you set this to false then the pagination and the show rows selector will not be called.

DEFAULT: true

data-tc-sort

If you set this to false then the th of the table will not allow you to order the table.

DEFAULT: true

data-tc-search

If you set this to false then the search will not be called.

DEFAULT: true
