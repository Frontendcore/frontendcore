Attributes

data-tc-type

Type of stats (pie, bar,area or line)

DEFAULT: line

data-tc-width

Width of the stats and the table in px

DEFAULT: autowidth based on parent element

data-tc-height

Height of the stats and the table in px

DEFAULT: 300

data-tc-table

Defines the position of the table (up, down, modal or hide)

DEFAULT: modal
