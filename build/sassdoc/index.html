    filter-blur 

    @mixin filter-blur($radius: $default-border-radius, $hardware: false) { ... }

    Description

    Blur Filter
    Applies blur on DOM elements. Fully compatible with all browsers for blur effect on images

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $radius

    Size in pixels of the blur effect

    		 Number $default-border-radius
    $hardware

    As blur requires a lot of ressources, with this variables a seemless 3d scale is applied to force hardware acceleration

    		 Boolean false

    font-size 

    @mixin font-size($size-value: 1.6, $line-height: true, $force-line-height: false) { ... }

    Description

    Font size
    Mixin to put a font-size and its correct line-height in rems (with fallback in px) into an element.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $size-value

    Size value of the font.

    		 Value 1.6
    $line-height

    If false, line-height property is not added.

    		 Boolean true
    $force-line-height

    If true, force line height to be the same as first parameter passed.

    		 Boolean false
